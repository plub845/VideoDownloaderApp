# VideoDownloaderApp

แอปพลิเคชัน Windows สำหรับดาวน์โหลดวิดีโอหรือเสียงจาก YouTube โดยใช้ `yt-dlp`และ`ffmpeg`

---

## 🎯 คุณสมบัติ

- ดาวน์โหลดวิดีโอเป็น `.mp4` หรือเสียงเป็น `.mp3`
- รองรับลิงก์ YouTube ที่มีพารามิเตอร์ต่าง ๆ
- เลือกโฟลเดอร์ปลายทางได้ตามต้องการ
- แสดงความคืบหน้าการดาวน์โหลดผ่าน Progress Bar
- รองรับการกำหนดคำสั่งพิเศษเพิ่มเติม (Custom Options)
- รองรับลิงก์ `.m3u8` สำหรับสตรีม (เวอร์ชันทดสอบ/เบต้า)

---

## 🛠️ วิธีใช้งาน

1. เปิดไฟล์ `VideoDownloaderApp.exe` ที่อยู่ในโฟลเดอร์ `main`
2. วางลิงก์ YouTube หรือ `.m3u8` ที่ต้องการดาวน์โหลด
3. เลือกประเภทไฟล์ที่ต้องการ: MP4 หรือ MP3
4. เลือกโฟลเดอร์สำหรับบันทึกไฟล์
5. กดปุ่ม **Download** แล้วรอจนการดาวน์โหลดเสร็จสิ้น

---

## ⚙️ วิธี build จากซอร์สโค้ด

1. เปิดโฟลเดอร์โปรเจกต์นี้ใน **Microsoft Visual Studio**
2. เปิดไฟล์ `.sln`
3. เลือก Build แบบ `Release`
4. จะได้ไฟล์ `.exe` สำหรับใช้งานในโฟลเดอร์ `bin\Release`

---

## 📌 หมายเหตุ

- โปรแกรมนี้ต้องใช้ `yt-dlp.exe` และ `ffmpeg.exe` (ควรวางไว้ในโฟลเดอร์เดียวกับ `.exe`)
- ช่อง "Custom Options" ใช้ใส่คำสั่งเพิ่มเติมของ yt-dlp เช่น `--playlist-items 1-5`
- **ห้ามกรอกข้อมูลใด ๆ ลงใต้ตัวหลังสือที่เป็นสีฟ้า**(ใช้สำหรับสถานะภายในโปรแกรมเท่านั้น)
